# Release Notes







## [v0.0.2](https://github.com/studioraz/test-magento2-cloudflare/releases/tag/0.0.2) — 2026-03-29

Initial release.

### Features

- Enable or disable Cloudflare integration via Magento admin.
- Configure image optimization settings with quality and fit options.
- Automatic cache management and purge operations using the Cloudflare API.
- Customizable paths to bypass cache for specific URLs.
- Tools for debugging Cloudflare Full Page Cache operations.
- Worker configuration settings for advanced cache control parameters.
